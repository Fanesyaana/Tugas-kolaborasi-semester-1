2x + 3y + 2z = 460.000 (pers 3)
II. Eliminasi persamaan 2 dan 1 x + y + 4z = 500.000
x + y + 2z = 300.000
2z = 200.000
z = 100.000
III. Substitusikan nilai z ke pers 1 dan 3 x + y + 2(100.000) = 300.000
x + y + 200.000 = 300.000
x + y = 100.000 (pers 4)

2x + 3y + 2(100.000) = 460.000
2x + 3y + 200.000 = 460.000
2x + 3y = 260.000 (pers 5)
IV. Eliminasi persamaan 4 dan 5 2x + 2y = 200.000 (pers 4) -> di kali 2
2x + 3y = 260.000 => y = 60.000
V. Substitusikan nilai y ke pers 4 x + 60.000 = 100.000
x = 40.000

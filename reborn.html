LANGKAH - LANGKAH RUMUS
I. Membuat model matematika dari soal cerita X = RAK, Y = MUG, Z = TAS
x + y + z = 105.000 (pers 1)
2x + 2y + z = 180.000 (pers 2)
3x + y + 2z = 235.000 (pers 3)
II. Eliminasi Z dari pers 1 dan 2 x + y + z = 105.000
2x + 2y + z = 180.000
Kurangi Pers (1) dari Pers (2):
(2x - x) + (2y - y) + (z - z) = 180.000 - 105.000 $
x + y = 75.000 (pers 4)
III. Eliminasi Z dari pers 1 dan 3 Kalikan Pers (1) dengan 2: 2x + 2y + 2z = 210.000 $
Kurangi dari Pers (3):
(3x - 2x) + (y - 2y) + (2z - 2z) = 235.000 - 210.000 $
x - y = 25.000 (pers 5)
IV. Eliminasi persamaan 4 dan 5 x + y = 75.000
x - y = 25.000
Kurangi kedua persamaan:
(x - x) + (y - (-y)) = 75.000 - 25.000
2y = 50.000 $ y = 25.000$**
V. Substitusikan nilai y ke pers 4 x + y = 75.000
x + 25.000 = 75.000
x = 75.000 - 25.000 x = 50.000$**
VI. Substitusikan nilai x dan y ke pers 1 x + y + z = 105.000
50.000 + 25.000 + z = 105.000
75.000 + z = 105.000
z = 105.000 - 75.000 z = 30.000
VII. Himpunan Penyelesaian (Harga Akhir) Harga RAK (x) = 50.000
Harga MUG (y) = 25.000
Harga TAS (z) = 30.000
HP = {(50.000, 25.000, 30.000)}
